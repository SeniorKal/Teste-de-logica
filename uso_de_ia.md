# Uso de IA

## Pergunta 1

Utilizei o ChatGPT para discutir diferentes estratégias de resolução.

Inicialmente implementei uma solução baseada em troca (swap), porém percebi que ela não preservava a ordem relativa dos demais elementos.

Após discutir o problema, optei por implementar uma solução utilizando deslocamento dos elementos dentro do próprio array.

Todo o código foi implementado manualmente, testado localmente e ajustado após validação dos resultados.
